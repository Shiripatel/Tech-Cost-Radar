CFO Executive Summary Brief

AI Costs are escalating quickly. This briefing summarizes cost trends, optimizations, audit guidelines, and vendor overlap opportunities for Q3.

[+] Key Overrun Drivers: The Engineering team's transition to Claude 3.5 Sonnet has increased prompt token depth by 45%. A large amount of this is redundant system prompts occurring in parallel test environments.


[+] Recommendation: Implement a semantic caching gateway (for instance, Helicone or LiteLLM gateway layers) to prevent sending duplicate codebase prompts to LLMs. This could cut Engineering AI costs by $12,000 monthly.


[-] Contract Renewals: The ChatGPT Enterprise agreement (100 seats) is up for review in 45 days. We currently see an average seat utilization of only 60%. We can safely down-scale to 60 seats or transition inactive users to OpenAI API keys via local chatbots.

Live Data Integration Guide

To move from simulated data to production, connect AI Cost Radar to your enterprise infrastructure using the methods below.

[+] Connectors Ready
4 Sources
[+] Auth Method
OAuth 2.0 / API Key
[+] Refresh Cadence
Every 15 min

[1] AI API Gateway — LiteLLM / Helicone / Portkey

Captures every LLM API call with token counts, latencies, and cost per request. This is the primary data feed for AI spend analytics.

Detail Value
Endpoint GET /api/v1/spend/logs
Auth Bearer token (API key from gateway admin panel)
Data fields provided timestamp, user_id, department, vendor, model, prompt_tokens, completion_tokens, cost_usd, purpose_tag
Setup time ~15 minutes (add proxy key + enable logging) 
# Example: Pull last 24h spend from LiteLLM
curl -H "Authorization: Bearer $LITELLM_KEY" \
  https://your-proxy.example.com/api/v1/spend/logs?start=2026-07-16

[2] SaaS Asset Manager — Zylo / Productiv / ServiceNow SAM

Exports license inventories, seat utilization metrics, and renewal dates. Used for vendor overlap detection and waste analysis.

Detail Value
Endpoint GET /api/v1/assets/licenses
Auth OAuth 2.0 (service account credentials)
Data fields provided vendor_name, license_name, allocated_seats, active_seats_30d, unit_cost_usd_monthly, category
Setup time ~30 minutes (configure OAuth app + map fields)

[3] Cloud Billing — AWS Cost Explorer / Azure Cost Mgmt / GCP Billing

Pulls infrastructure costs (compute, storage, networking) to feed the IT Run Rate tile and forecast model.

Detail Value
Endpoint (AWS) POST ce:GetCostAndUsage
Auth IAM role with ce:GetCostAndUsage permission
Data fields provided billing_period, resource_provider, service_category, allocated_department, gross_spend_usd
Setup time ~20 minutes (create IAM role + activate Cost Explorer)

[4] Business Attribution — Custom Webhook / CSV Upload

Finance teams post project ROI metrics to measure whether AI spend translates into revenue gains. This can be a simple CSV upload or webhook.

Detail Value
Endpoint POST /api/v1/roi/attribution or manual CSV upload
Auth API key or SSO-authenticated file upload
Data fields provided project_name, associated_ai_spend_usd, monthly_revenue_attributed_usd, equivalent_non_ai_cost_usd
Setup time ~10 minutes (CSV template) or ~45 minutes (webhook integration)
